üÉè Final Review Flashcards

What are the three main ensemble learning paradigms?
Bagging (Bootstrap Aggregating), Boosting, and Stacking. Bagging trains models in parallel, boosting trains sequentially correcting errors, stacking combines predictions with a meta-learner.
Why do random forests work well?
They reduce overfitting through bagging and decorrelate trees via random feature selection. Each tree sees different data and features, making the ensemble more robust than individual trees.
What's the key innovation of gradient boosting?
It uses gradient descent in function space. Each new tree predicts the negative gradient (pseudo-residuals) of the loss function, effectively optimizing any differentiable loss function.
When should you use bagging vs boosting?
Use bagging (random forests) when you have high-variance models and want parallel training. Use boosting when you want to minimize bias and don't mind sequential training. Boosting often achieves better accuracy but may overfit.
