Key Concepts to Cover:

  • Decision Trees: Recursive binary splitting, impurity measures (Gini, Entropy)
  • Tree Pruning: Preventing overfitting through pre-pruning and post-pruning
  • Random Forests: Ensemble of decision trees with bagging
  • Bootstrap Sampling: Random sampling with replacement
  • Feature Importance: Which features contribute most to predictions
  • Out-of-Bag (OOB) Error: Built-in validation for random forests

üéØ Mini-Task

Goal: Implement decision trees and random forests for classification

  1. Install required libraries (scikit-learn, matplotlib)
  2. Load a classification dataset (e.g., iris or make_classification)
  3. Split data into training and testing sets
  4. Train a DecisionTreeClassifier and evaluate performance
  5. Train a RandomForestClassifier and compare results
  6. Examine feature importance from the random forest
  7. Visualize a decision tree (optional)

Sample Code:

import numpy as np
from sklearn.datasets import load_iris, make_classification
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier, plot_tree
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score, classification_report
import matplotlib.pyplot as plt

# Load iris dataset (or create synthetic data)
iris = load_iris()
X, y = iris.data, iris.target
feature_names = iris.feature_names
target_names = iris.target_names

# Split the data
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# Train Decision Tree
dt_model = DecisionTreeClassifier(max_depth=3, random_state=42)
dt_model.fit(X_train, y_train)
dt_pred = dt_model.predict(X_test)
dt_accuracy = accuracy_score(y_test, dt_pred)

print(f"Decision Tree Accuracy: {dt_accuracy:.3f}")
print("\nDecision Tree Classification Report:")
print(classification_report(y_test, dt_pred, target_names=target_names))

# Train Random Forest
rf_model = RandomForestClassifier(n_estimators=100, max_depth=3, random_state=42, oob_score=True)
rf_model.fit(X_train, y_train)
rf_pred = rf_model.predict(X_test)
rf_accuracy = accuracy_score(y_test, rf_pred)

print(f"\nRandom Forest Accuracy: {rf_accuracy:.3f}")
print(f"Random Forest OOB Score: {rf_model.oob_score_:.3f}")
print("\nRandom Forest Classification Report:")
print(classification_report(y_test, rf_pred, target_names=target_names))

# Feature importance
feature_importance = rf_model.feature_importances_
print("\nFeature Importance:")
for name, importance in zip(feature_names, feature_importance):
    print(f"{name}: {importance:.3f}")

# Visualize decision tree (optional - requires matplotlib)
plt.figure(figsize=(12, 8))
plot_tree(dt_model, feature_names=feature_names, class_names=target_names, 
          filled=True, rounded=True, fontsize=10)
plt.title("Decision Tree Visualization")
plt.show()
