üéØ Mini-Task

Goal: Load a sample dataset and explore its structure

  1. Install pandas: pip install pandas
  2. Download a simple dataset (e.g., Iris flowers from sklearn)
  3. Load the data into a pandas DataFrame
  4. Use exploration methods: head(), describe(), info()
  5. Answer: How many rows/columns? What data types? Any missing values?

Sample Code:

import pandas as pd
from sklearn.datasets import load_iris

# Load sample data
iris = load_iris()
df = pd.DataFrame(iris.data, columns=iris.feature_names)
df['target'] = iris.target

# Explore the data
print("First 5 rows:")
print(df.head())
print("\nData info:")
print(df.info())
print("\nSummary statistics:")
print(df.describe())
