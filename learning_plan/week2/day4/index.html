üÉè Key Concepts Flashcards

What's the difference between precision and recall?
Precision = TP/(TP+FP) measures accuracy of positive predictions. Recall = TP/(TP+FN) measures how many actual positives were found. High precision = few false positives, high recall = few false negatives.
When should you use F1-score instead of accuracy?
Use F1-score for imbalanced datasets where accuracy can be misleading. F1 = 2*(precision*recall)/(precision+recall) balances both precision and recall.
What does AUC-ROC measure and when is it useful?
AUC measures the area under the ROC curve, representing model's ability to distinguish classes. Useful for comparing models and choosing classification thresholds, especially with imbalanced data.
How do you choose between MSE and MAE for regression evaluation?
Use MSE when large errors are particularly bad (squares large errors). Use MAE when all errors should be treated equally. MSE is more sensitive to outliers.
